HOW TO FIX USING EZGIF (Online - No Software Needed):
=====================================================

1. Go to: https://ezgif.com/optimize
2. Click "Choose File" and select the first GIF from the list above
3. Scroll down to find the "Disposal method" dropdown
4. Change it from "Do not dispose" or "Unspecified" to "Restore to background"
5. Click "Save optimized GIF"
6. Download the fixed GIF and replace the original file
7. Repeat for all 8 GIF files

ALTERNATIVE: If you have GIMP installed locally:

2. Go to: File > Export As
3. Export as GIF
4. In the GIF export dialog, find "Disposal" settings
5. Set frame disposal to "Replace" or "Restore to Background"
6. Save

this properly, but the GIF files themselves have the wrong settings.
